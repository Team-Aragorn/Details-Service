## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

### Seed the DB

```sh
npm run seed
```
When you see 'DONE SEEDING' Press CTRL-C

### Starting the App
Open two terminal windows and run the following:

Build the webpack bundle

```sh
npm run react-dev
```

Start the App

```sh
npm start
```
